4 — Cross-Cutting Systems
Code Graph
SQLite database of files, symbols, dependencies, modules, capabilities. Queried by factory (context + risk), verification (contracts + cycles), and updater (incremental). Snapshots saved before/after execution.
forge-graph/builder.js
forge-graph/query.js
forge-graph/updater.js
forge-graph/snapshot.js
Session Ledger
Markdown file tracking execution state, decisions, warnings, user preferences. Written by wave execution, read by agent factory for knowledge propagation. Auto-compacts when too large, archived on phase complete.
forge-session/ledger.js
.forge/session/ledger.md
.forge/session/archive/
Unified Config
Single source of truth. Merge order: defaults ← ~/.forge/config.json (global) ← .forge/config.json (project). 12 sections: graph, execution, containers, agents, verification, knowledge, impact_analysis, session, display, git, system, project.
forge-config/config.js
forge-config/doctor.js
.forge/config.json
System Graph
Cross-repo dependency tracking. Services, interfaces, dependencies. Provides cross-repo context to agent factory (consumer warnings), verification L7 (contract drift), and parallel planner (cross-repo DAG edges).
forge-system/query.js
forge-system/sync.js
forge-system/dashboard.js
forge-verify/contract-layer.js
Knowledge Base
Persistent learnings that survive across milestones. Auto-promoted from ledger on archive (decisions, warnings, pitfalls, preferences). Filtered by module relevance and injected into agent system prompts. CLI: knowledge list/add/prune/promote.
forge-session/knowledge.js
.forge/knowledge/learnings.json
forge-config/config.js (knowledge section)
Architecture Review
Two-level architectural fitness checking. Plan-checker Dimension 8: validates plans against ARCHITECTURE.md & CONVENTIONS.md (suggestions, non-blocking). Verification Layer 8: optional agent-based code review against documented conventions (off by default).
agents/forge-plan-checker.md (Dim 8)
forge-verify/engine.js (Layer 8)
.planning/codebase/ARCHITECTURE.md
.planning/codebase/CONVENTIONS.md
